Underrail Feat Generator

Live Website

A web application that takes nearly every feat in the game Underrail, compares it to the skill and stat values you provide and generates relevant feats, which it returns to you. The backend is written in Go, and the front end uses Vue3 with Pinia for state management. For the database, I use the relational database MariaDB.
